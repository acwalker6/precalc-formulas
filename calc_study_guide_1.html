Core Differentiation Rules

// Fundamental Rules — Memorize Cold
Constant Rule d/dx [c] = 0
Power Rule d/dx [xⁿ] = n·xⁿ⁻¹ Works for any real n, including fractions and negatives
Constant Multiple d/dx [c·f(x)] = c·f'(x)
Sum / Difference d/dx [f ± g] = f' ± g'
Product Rule d/dx [f·g] = f'g + fg' "First times deriv of second, plus second times deriv of first"
Quotient Rule d/dx [f/g] = (f'g − fg') / g² Lo·dHi minus Hi·dLo over Lo-squared. g(x) ≠ 0
Chain Rule d/dx [f(g(x))] = f'(g(x))·g'(x) Outer derivative × inner derivative. The single most tested rule.
Chain Rule (Leibniz) dy/dx = dy/du · du/dx u = g(x), y = f(u)
⚠ Chain Rule Trap You MUST apply the chain rule every time the inner function is anything other than bare x. d/dx[sin(3x²)] = cos(3x²)·6x, not just cos(3x²).
02 ——

Trigonometric Derivatives

Sine d/dx [sin x] = cos x
Cosine d/dx [cos x] = −sin x
Tangent d/dx [tan x] = sec²x
Cotangent d/dx [cot x] = −csc²x
Secant d/dx [sec x] = sec x · tan x
Cosecant d/dx [csc x] = −csc x · cot x
Pattern The "co-" functions (cos, cot, csc) all carry a negative sign. The "non-co" functions (sin, tan, sec) are all positive. Lock this in and you will never misremember.
03 ——

Logarithmic & Exponential Derivatives

Natural Log d/dx [ln x] = 1/x x > 0
ln|x| (both sides) d/dx [ln|x|] = 1/x x ≠ 0; valid for negative x too
Chain Rule on ln d/dx [ln(u)] = u'/u Where u = g(x). This is the key form for almost every ln problem.
Log base a d/dx [log_a(x)] = 1/(x·ln a)
e^x d/dx [eˣ] = eˣ The only function that is its own derivative.
Chain Rule on e^u d/dx [e^u] = e^u · u' u = g(x); the exponent's derivative multiplies out front
a^x (any base) d/dx [aˣ] = aˣ · ln(a)
a^u (chain) d/dx [a^u] = a^u · ln(a) · u'
// Logarithmic Differentiation — When to Use It
Technique When you see a variable raised to a variable power (like x^x or x^(sin x)), take ln of both sides, differentiate implicitly, then solve for y'. This sidesteps the chain rule nightmare entirely.
Log Differentiation — Example Template y = x^x
ln y = x·ln x
y'/y = ln x + x·(1/x) = ln x + 1
y' = x^x (ln x + 1)
// Key Log Properties (Algebra Before Differentiation)
ln(ab) = ln a + ln b
ln(a/b) = ln a − ln b
ln(aⁿ) = n·ln a
ln(eˣ) = x  |  e^(ln x) = x

Inverse Trigonometric Derivatives

Note on Notation arcsin = sin⁻¹, arccos = cos⁻¹, arctan = tan⁻¹. These are NOT reciprocals; the -1 means inverse function. Every formula below pairs with its "co-" twin which is identical but negative.
arcsin x d/dx [arcsin x] = 1 / √(1−x²) Domain: |x| < 1
arccos x d/dx [arccos x] = −1 / √(1−x²) Exact negative of arcsin
arctan x d/dx [arctan x] = 1 / (1+x²) Most frequently tested. No square root.
arccot x d/dx [arccot x] = −1 / (1+x²)
arcsec x d/dx [arcsec x] = 1 / (|x|√(x²−1)) |x| > 1
arccsc x d/dx [arccsc x] = −1 / (|x|√(x²−1))
Chain Rule Always Applies d/dx[arctan(u)] = u' / (1+u²). The u' in the numerator is what students consistently forget under exam pressure.
05 ——

Implicit Differentiation

// The Process
  • 1Differentiate both sides with respect to x.
  • 2Every time you differentiate a term with y in it, multiply by dy/dx (chain rule — y is a function of x).
  • 3Collect all dy/dx terms on one side of the equation.
  • 4Factor out dy/dx and divide to isolate it.
Example: circle x² + y² = r² 2x + 2y·dy/dx = 0
dy/dx = −x/y
Second Derivative Implicitly To find d²y/dx², differentiate dy/dx implicitly again and substitute the expression for dy/dx back in. Messy, but methodical.
06 ——

Related Rates

  • 1Draw a diagram. Label every quantity that changes with time as a variable.
  • 2Write an equation relating those variables (geometry formulas, Pythagorean theorem, similar triangles, etc.).
  • 3Differentiate both sides with respect to time t (implicit differentiation).
  • 4Plug in all known values and solve for the unknown rate.
⚠ Common Mistake Do NOT substitute specific values for variables before differentiating. Differentiate the general equation first, then substitute. Substituting too early kills the dt terms.
// Frequently Appearing Related Rates Setups
Pythagorean (ladder, shadow) x² + y² = z²
2x·ẋ + 2y·ẏ = 2z·ż
Circle area expanding A = πr²
dA/dt = 2πr · dr/dt
Sphere volume V = (4/3)πr³
dV/dt = 4πr² · dr/dt
Cone volume (variable r,h) V = (1/3)πr²h
Use similar triangles if r/h = const

Chapter 4 · Theorems & Derivative Tests

Extreme Value Theorem (EVT)
If f is continuous on a closed interval [a, b], then f attains both an absolute maximum and an absolute minimum on [a, b]. The key constraint is closed and bounded, not open.
Fermat's Theorem
If f has a local max or min at c, and f'(c) exists, then f'(c) = 0. Critical numbers: where f'(c) = 0 OR f'(c) does not exist.
Rolle's Theorem
If f is continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then there exists at least one c in (a,b) such that f'(c) = 0.
Mean Value Theorem (MVT)
If f is continuous on [a,b] and differentiable on (a,b), then there exists c in (a,b) such that f'(c) = [f(b) − f(a)] / (b − a). The instantaneous rate of change equals the average rate at some point. This is one of the most important theoretical results in all of calculus.
// First Derivative Test
Behavior of f' around critical point cConclusion
f' changes + → −Local Maximum at c
f' changes − → +Local Minimum at c
f' does not change signNo local extremum (inflection point possible)
// Second Derivative Test
ConditionConclusion
f'(c) = 0 and f''(c) > 0Local Minimum at c (concave up)
f'(c) = 0 and f''(c) < 0Local Maximum at c (concave down)
f'(c) = 0 and f''(c) = 0Test fails. Use 1st derivative test.
// Concavity & Inflection Points
Concave Up f''(x) > 0 on interval Graph curves upward like a bowl. f' is increasing.
Concave Down f''(x) < 0 on interval Graph curves downward like a hill. f' is decreasing.
Inflection Point Where f''(x) = 0 AND concavity changes f''=0 alone is NOT sufficient. The sign must change.
// Finding Absolute Extrema on [a, b] — Closed Interval Method
  • 1Find all critical numbers of f in the open interval (a, b).
  • 2Evaluate f at each critical number and at the endpoints a and b.
  • 3The largest value is the absolute max; the smallest is the absolute min.

L'Hôpital's Rule

L'Hôpital's Rule
If lim f(x)/g(x) produces the indeterminate form 0/0 or ±∞/±∞, and if f and g are differentiable near a, then:

lim [f(x)/g(x)] = lim [f'(x)/g'(x)]

Apply this as many times as necessary until the limit resolves. You are differentiating numerator and denominator independently, not using the quotient rule.
// Indeterminate Forms & How to Handle Them
FormStrategy
0/0 , ∞/∞Apply L'Hôpital directly.
0 · ∞Rewrite as 0/(1/∞) or ∞/(1/0) to get a fraction, then apply L'Hôpital.
∞ − ∞Combine into a single fraction, then apply L'Hôpital.
1^∞, 0⁰, ∞⁰Take ln of the limit expression, evaluate, then exponentiate the result.
⚠ When NOT to Use L'Hôpital If the limit is NOT indeterminate (e.g., it already equals a finite number or clean infinity), L'Hôpital does not apply. Using it anyway produces wrong answers.

Curve Sketching (The Full Checklist)

  • ADomain: Where is f(x) defined? Note restrictions.
  • BIntercepts: x-intercepts (set y=0), y-intercept (set x=0).
  • CSymmetry: Even (f(-x)=f(x), symmetric about y-axis), Odd (f(-x)=-f(x), rotational symmetry), or neither.
  • DAsymptotes: Horizontal (lim as x→±∞), Vertical (where denominator=0 or function blows up), Oblique (if degree of numerator exceeds denominator by 1, do polynomial long division).
  • EIncreasing/Decreasing: Find f'. Solve f'(x)=0. Build sign chart. f'>0 means increasing; f'<0 means decreasing.
  • FLocal Extrema: Apply 1st or 2nd derivative test at critical numbers.
  • GConcavity: Find f''. Solve f''(x)=0. Build sign chart. Note inflection points where concavity changes.
  • HSketch: Plot intercepts, asymptotes, and extrema. Connect using concavity information.

Optimization Problems

  • 1Read the problem carefully. Identify what is to be maximized or minimized. Name it Q.
  • 2Write an equation for Q in terms of the relevant variables.
  • 3Use a constraint equation to reduce Q to a function of ONE variable.
  • 4Find the domain of Q (often a closed or half-open interval from the physical context).
  • 5Differentiate Q, set Q'=0, find critical numbers.
  • 6Use the closed interval method OR first/second derivative test to confirm it is a max or min.
  • 7Answer the question; include units.
Tip The constraint is often hidden in the word problem as a fixed total: perimeter, cost, surface area, volume, or total length. Identify it immediately and write it down before doing anything else.

Antiderivatives (§4.9)

What is an Antiderivative? F is an antiderivative of f if F'(x) = f(x). The general antiderivative is F(x) + C, where C is an arbitrary constant. This bridges Chapter 4 to integration.
Power Rule (reverse) ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C n ≠ −1
n = −1 exception ∫ 1/x dx = ln|x| + C
∫ eˣ dx = eˣ + C
∫ sin x dx = −cos x + C
∫ cos x dx = sin x + C
∫ sec²x dx = tan x + C
∫ csc²x dx = −cot x + C
∫ sec x tan x dx = sec x + C
∫ csc x cot x dx = −csc x + C
∫ 1/√(1−x²) dx = arcsin x + C
∫ 1/(1+x²) dx = arctan x + C
// Rectilinear Motion (Position, Velocity, Acceleration)
a(t) = v'(t) = s''(t) Acceleration is derivative of velocity, 2nd derivative of position
v(t) = ∫ a(t) dt Integrate acceleration to get velocity; use initial condition to find C
s(t) = ∫ v(t) dt Integrate velocity to get position

Area & Volume of Common Shapes

// 2D Areas
○ Circle
Area: A = πr²
Circumference: C = 2πr
▭ Rectangle
Area: A = lw
Perimeter: P = 2l + 2w
△ Triangle
Area: A = ½bh
Perimeter: P = a + b + c
▱ Trapezoid
Area: A = ½(b₁ + b₂)h
b₁, b₂ are the parallel bases
◇ Parallelogram
Area: A = bh
h = perpendicular height (not side length)
⌒ Sector of Circle
Area: A = ½r²θ
Arc Length: s = rθ
θ must be in radians
// 3D Volumes & Surface Areas
Sphere
Volume: V = (4/3)πr³
Surface Area: SA = 4πr²
Cylinder
Volume: V = πr²h
Lateral SA: 2πrh
Total SA: 2πr² + 2πrh
Cone
Volume: V = (1/3)πr²h
Lateral SA: πrl  (l = slant height)
Total SA: πr² + πrl
Rectangular Box
Volume: V = lwh
Surface Area: SA = 2(lw + lh + wh)
Square Pyramid
Volume: V = (1/3)Bh
B = area of base, h = height
General Prism
Volume: V = Bh
B = base area, h = height
