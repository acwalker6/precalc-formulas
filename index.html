Coordinate Geometry
Core Formula
Distance Formula
d = √[(x₂ − x₁)² + (y₂ − y₁)²]
Derived from the Pythagorean theorem. The horizontal difference and vertical difference form the legs; the distance is the hypotenuse.
Core Formula
Midpoint Formula
M = ((x₁ + x₂)/2, (y₁ + y₂)/2)
Average the x's, average the y's. To reverse-engineer a missing endpoint: x₂ = 2Mx − x₁.
Core Formula
Slope
m = (y₂ − y₁) / (x₂ − x₁)
Rise over run. Undefined for vertical lines (Δx = 0). Zero for horizontal lines (Δy = 0).
Equations of Lines
Form
Slope-Intercept
y = mx + b
m = slope, b = y-intercept. Best for graphing quickly.
Form
Point-Slope
y − y₁ = m(x − x₁)
Use when given a slope and any point. The workhorse form for exam problems.
Form
Standard Form
Ax + By = C  (A > 0, integers)
x-int: set y=0 → x = C/A. y-int: set x=0 → y = C/B.
Special Lines
Vertical & Horizontal
Vertical: x = a  |  Horizontal: y = b
Vertical through (−3, 7) → x = −3. Horizontal through (5, −2) → y = −2.
RelationshipSlope RuleQuick Check
Parallelm₁ = m₂Same slope, different intercept
Perpendicularm₁ · m₂ = −1Flip fraction, negate sign
Same linem₁ = m₂ AND b₁ = b₂Identical when simplified
🧠 Memorize
Perpendicular slope of 2/3 → −3/2. Perpendicular slope of −5 → 1/5. Flip and negate, always.
Circles
Core Formula
Standard Form
(x − h)² + (y − k)² = r²
Center: (h, k). Radius: r. Signs inside parentheses are OPPOSITE the center coordinates.
Process
Completing the Square
x² + bx → (x + b/2)² − (b/2)²
Half the linear coefficient, square it, add to both sides. Works for both circle equations and quadratics.
Test
Point vs Circle
Plug (x,y) into left side. Compare to r².
= r² → on circle. < r² → inside. > r² → outside.
Shortcut
Circle from Diameter
Center = midpoint of endpoints
r = ½ · distance between endpoints
Two formulas you already know, combined.
Functions & Operations
Operations
Arithmetic Combos
(f ± g)(x) = f(x) ± g(x)
(fg)(x) = f(x)·g(x)
(f/g)(x) = f(x)/g(x), g(x) ≠ 0
For division, exclude values where g(x) = 0 from the domain.
Core Concept
Composition
(f ∘ g)(x) = f(g(x))
g runs first (inner function), then f (outer function). Order matters: f(g(x)) ≠ g(f(x)) in general.
Core Formula
Difference Quotient
[f(x + h) − f(x)] / h
Steps: (1) Compute f(x+h). (2) Subtract f(x). (3) Divide by h. (4) Simplify until h cancels. This is the precalculus preview of the derivative.
Core Formula
Average Rate of Change
[f(b) − f(a)] / (b − a)
Slope of the secant line between two points on the curve. Same structure as the difference quotient with specific values.
🧠 Common Difference Quotients to Memorize
f(x) = mx + b → DQ = m (constant)
f(x) = x² → DQ = 2x + h
f(x) = 1/x → DQ = −1/[x(x+h)]
f(x) = √x → DQ = 1/[√(x+h) + √x] (rationalize the numerator)

Symmetry & Even/Odd
TestReplacementIf unchangedFunction Type
Y-axis symmetryx → −xSame equationEven
X-axis symmetryy → −ySame equation(not a function)

Rule
Even Function
f(−x) = f(x)
Y-axis mirror. If (a, b) is on the graph, so is (−a, b). All even-powered terms.
Rule
Odd Function
f(−x) = −f(x)
Origin rotation. If (a, b) is on the graph, so is (−a, −b). All odd-powered terms.
🧠 Quick Checks
x⁴ − 2x² → all even powers → even
x³ − x → all odd powers → odd
x² + x → mixed powers → neither
|x³| → absolute value of odd = even

For y = −f(x − 1) + 2: Right 1 → reflect over x-axis → up 2.
Piecewise
Graphing Rules
Different formula per interval
(1) Identify domain intervals. (2) Graph each piece on its interval only. (3) Closed circle ● = included. Open circle ○ = excluded at boundaries.
Key Rule
Horizontal Shift Trap
f(x − 3) shifts RIGHT
f(x + 3) shifts LEFT
The sign inside is opposite the direction. "Inside is a lie."
Quadratic Functions
Form
Standard Form
f(x) = ax² + bx + c
a determines direction. b and c determine position. Vertex: h = −b/(2a).
Form
Vertex Form
f(x) = a(x − h)² + k
Vertex: (h, k). Axis of symmetry: x = h. Obtained by completing the square.
Shortcut
Vertex from Standard
h = −b/(2a)
k = f(h)
Find h first, then plug h back into f to get k. Faster than completing the square.
Rule
Max vs Min
a > 0 → minimum (opens up)
a < 0 → maximum (opens down)
Positive a = happy face (min). Negative a = sad face (max). The vertex is always the extreme value.
🧠 Completing the Square Process
(1) Factor out a from x² and x terms → (2) Half the linear coefficient, square it → (3) Add/subtract inside parentheses → (4) Factor the perfect square, distribute the subtracted value out.

Watch: When factoring out a negative, the sign of the value you "move out" flips. −2(x²+2x+1−1)+5 → the −1 becomes +2 (multiplied by −2).
Quadratic Inequalities
Process
Sign Chart Method
(1) Solve = 0 for roots
(2) Plot on number line
(3) Test each interval
(4) Write interval notation
≤ or ≥ → use brackets [ ]. < or > → use parentheses ( ).
Pattern
Quick Solve (a > 0)
(x−a)(x−b) ≥ 0 → (−∞,a] ∪ [b,∞)
(x−a)(x−b) ≤ 0 → [a,b]
Opens up: positive outside roots, negative between roots. Flip for a < 0.
Trick Question
No Real Roots
b² − 4ac < 0
If discriminant < 0 and a > 0 → always positive → > 0 is (−∞,∞) and < 0 is ∅.
If a < 0 → always negative → flip the answers.
Formula
Discriminant
Δ = b² − 4ac
Δ > 0: two real roots. Δ = 0: one repeated root. Δ < 0: no real roots (complex).
🧠 Interval Notation Cheat
Brackets [ ] = endpoint included (≤ or ≥). Parentheses ( ) = endpoint excluded (< or >).
∞ always gets parentheses. You can't "include" infinity.
